                        TROUBLESHOOTING GUIDE
???????????????????????????????????????????????????????????????????????????????

PROBLEM: Camera still not opening
?????????????????????????????????????????????????????????????????????????????
Solution 1: Check camera connection
  • Ensure WebCam is connected (USB or built-in)
  • Check if LED light is on
  • Try unplugging/replugging

Solution 2: Close conflicting applications
  • Close: Skype, Teams, Zoom, Discord, OBS
  • Close: Windows Camera app
  • Close: Any other camera-using software

Solution 3: Grant Windows permissions
  • Settings ? Privacy & Security ? Camera
  • Check Python has camera access
  • Enable camera for Python

Solution 4: Restart application
  • Stop Flask server (Ctrl+C)
  • Wait 5 seconds
  • Start again: python app.py

Solution 5: Restart computer
  • This usually resolves camera driver issues


PROBLEM: Video stream appears but face detection not working
?????????????????????????????????????????????????????????????????????????????
Solution 1: Improve lighting
  • Turn on room lights
  • Avoid shadows on face
  • Face should be well-lit

Solution 2: Check camera view
  • Face should be clearly visible
  • Entire face in frame
  • Not too close or too far

Solution 3: Test with Windows Camera
  • Open Windows Camera app
  • Verify face detection works there
  • This confirms camera is working

Solution 4: Clean camera lens
  • Dust can interfere with detection
  • Gently wipe with soft cloth


PROBLEM: Beep sound not playing
?????????????????????????????????????????????????????????????????????????????
Solution 1: Check system volume
  • Speaker icon in taskbar
  • Unmute if muted
  • Increase volume

Solution 2: Test system sound
  • Play YouTube video
  • Verify audio works
  • Check speaker is connected

Note: This is non-critical - visual warnings still display


PROBLEM: Screenshots not saving
?????????????????????????????????????????????????????????????????????????????
Solution: Check directory permissions
  • Folder: static/captured_images/
  • Must be writable by Python
  • Check file permissions
  • Try running as administrator


PROBLEM: Login not working
?????????????????????????????????????????????????????????????????????????????
Solution 1: Clear browser cookies
  • Open DevTools (F12)
  • Application tab ? Cookies ? Clear all

Solution 2: Verify registration
  • Check user was registered successfully
  • Try registering again
  • Use different username

Solution 3: Check database
  • File: users.db
  • If corrupted, delete and restart
  • App will recreate database

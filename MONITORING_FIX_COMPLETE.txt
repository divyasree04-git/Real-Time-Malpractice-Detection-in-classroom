?????????????????????????????????????????????????????????????????????????????

Still showing "Exam Not Started"?
??????????????????????????????????????????????????????????????????????????????
1. Clear browser cache:
   • Ctrl + Shift + Delete
   • Clear all cookies and cache
   • Reload page

2. Hard refresh:
   • Ctrl + Shift + R (Windows)
   • Command + Shift + R (Mac)

3. Restart Flask:
   • Stop: Ctrl + C
   • Start: python app.py

4. Check browser console:
   • Press F12 to open Developer Tools
   • Go to Console tab
   • Look for JavaScript errors


Alert doesn't appear?
??????????????????????????????????????????????????????????????????????????????
1. Open browser console (F12)
2. Check for "Failed to start exam" message
3. Check Network tab for /start_class response
4. Verify server is running


Video feed doesn't show?
??????????????????????????????????????????????????????????????????????????????
1. Check camera is connected
2. Close other apps using camera
3. Restart Flask app
4. Check camera permissions in Windows Settings


No face detection?
??????????????????????????????????????????????????????????????????????????????
1. Ensure good lighting
2. Position face clearly in view
3. Keep 30-60 cm from camera
4. Clean camera lens if dusty

                    CAMERA TROUBLESHOOTING
???????????????????????????????????????????????????????????????????????????

Issue: "CAMERA ERROR - Check camera connection" appears
??????????????????????????????????????????????????????
Steps to fix:
1. Check if camera is plugged in (USB or built-in)
2. Close other apps using camera:
   - Skype, Teams, Zoom, Discord
   - OBS, VLC, Camera app
   - Any video call application
3. Restart the application:
   - Stop Flask server (Ctrl+C)
   - Start again: python app.py
4. Try a different USB port (if external camera)
5. Restart computer (if nothing else works)


Issue: Video stream shows "CAMERA FEED ERROR" during exam
?????????????????????????????????????????????????????????
Steps to fix:
1. This is temporary - app will auto-recover
2. Wait 1-2 seconds for stream to resume
3. If persists:
   - Stop exam ("Stop Exam" button)
   - Start exam again ("Start Exam" button)
4. If still fails:
   - Check USB connection
   - Close camera-using apps
   - Restart application


Issue: Camera works but face detection not working
??????????????????????????????????????????????????
Steps to fix:
1. Check lighting conditions:
   - Should be well-lit
   - Avoid backlighting
   - Position light source in front
2. Check distance:
   - Should be 30-60 cm away (1-2 feet)
   - Full face visible
3. Check angle:
   - Face should be straight to camera
   - Not extreme angles
4. Test with other camera app:
   - Windows Camera app
   - Verify face detection works there


Issue: Beep not working
??????????????????????
Steps to fix:
1. Check system volume is not muted
2. Check speaker is plugged in and working
3. Test system sounds:
   - Play a YouTube video
   - Verify audio works
4. This is non-critical - warnings still display visually

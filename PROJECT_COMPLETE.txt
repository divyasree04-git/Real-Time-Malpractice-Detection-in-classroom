                         TROUBLESHOOTING
???????????????????????????????????????????????????????????????????????????????

ISSUE: Python not found
SOLUTION: 
- Install Python 3.7+ from https://www.python.org/
- Check "Add Python to PATH" during installation
- Restart computer

ISSUE: Permission denied on batch files
SOLUTION:
- Right-click batch file
- Select "Run as Administrator"
- Or run from Command Prompt directly

ISSUE: Webcam not working
SOLUTION:
- Check camera is connected
- Close other camera-using apps
- Grant permissions to Python if prompted
- Check camera isn't in use by another application

ISSUE: Dependencies installation fails
SOLUTION:
- Try: pip install --upgrade pip
- Try: pip install -r requirements.txt (manual)
- Check internet connection
- Check Python version: python --version

ISSUE: Port 5000 in use
SOLUTION:
- Edit app.py (last line)
- Change: app.run(debug=True, port=5001)
- Save and run again

ISSUE: Database shows no saved users
SOLUTION:
- Previously fixed! Now uses CREATE TABLE IF NOT EXISTS
- Users are permanently saved
- Delete users.db to reset (will lose all user data)

                         CONFIGURATION OPTIONS
???????????????????????????????????????????????????????????????????????????????

To change inattention threshold:
- Edit: app.py
- Find: if inattention_duration >= 3.0:
- Change: 3.0 to desired seconds (e.g., 2.0 for 2 seconds)
- Save file
- Restart application

To change beep settings:
- Edit: app.py (in play_beep() function)
- Find: winsound.Beep(1000, 500)
- First param: frequency in Hz (1000 = 1kHz)
- Second param: duration in milliseconds (500 = 0.5 seconds)
- Save and restart

To change server port:
- Edit: app.py (last line)
- Find: app.run(debug=True)
- Change to: app.run(debug=True, port=5001)
- Save and restart

# Real Time Exam Malpractice Detection - Setup Guide

## Quick Start

### For First-Time Users (Recommended)
1. Double-click **`setup_and_run.bat`** 
   - Automatically installs all dependencies
   - Creates necessary folders
   - Starts the application

### For Subsequent Runs
- Double-click **`run_app.bat`** to run the application
- Or use command: `python app.py`

---

## Available Batch Files

### 1. **setup_and_run.bat** (RECOMMENDED for first run)
   - Checks Python installation
   - Installs all dependencies from requirements.txt
   - Creates static/captured_images folder
   - Starts Flask application
   - No additional steps needed

### 2. **install_dependencies.bat** (Install only)
   - Checks Python and pip
   - Installs dependencies
   - Useful if you only want to install without running

### 3. **run_app.bat** (Run only)
   - Starts the Flask application
   - Uses http://localhost:5000
   - Useful for subsequent runs after setup

---

## Requirements.txt

**File**: `requirements.txt`

**Contents**:
```
Flask==2.0.1
opencv-python==4.5.3.56
numpy==1.21.2
Werkzeug==2.0.1
```

**What each package does**:
- **Flask**: Web framework for the application interface
- **opencv-python**: Face detection and video processing
- **numpy**: Numerical computations
- **Werkzeug**: Password hashing and security

## Installation Steps

### Method 1: Automatic (Easiest)
```

```

## Troubleshooting

### "Python not found"
- Install Python 3.7+ from https://www.python.org/
- During installation, check "Add Python to PATH"
- Restart computer after installation

### "Module not found error"
- Run: `install_dependencies.bat`
- Or: `pip install -r requirements.txt`

### "Port 5000 already in use"
- Edit last line of app.py
- Change: `app.run(debug=True)` 
- To: `app.run(debug=True, port=5001)`

### "Webcam not working"
- Check if camera is connected
- Close other apps using camera
- Grant camera permissions to Python

### "Installation hangs"
- Press Ctrl+C to cancel
- Try: `pip install --upgrade pip`
- Then run install_dependencies.bat again

## System Configuration

### Inattention Threshold
Default: 3 seconds

To change, edit `app.py` line ~113:
```python
if inattention_duration >= 3.0:  # Change 3.0 to desired seconds
```

### Beep Sound Settings
Default: 1000 Hz for 500ms

To change, edit `app.py` line ~65:
```python
winsound.Beep(1000, 500)  # (frequency, duration_ms)
```

### Server Port
Default: 5000

To change, edit `app.py` last line:
```python
app.run(debug=True, port=5000)  # Change 5000 to desired port
```

?????????????????????????????????????????????????????????????????????????????
                    TROUBLESHOOTING
?????????????????????????????????????????????????????????????????????????????

ERROR: 'python' is not recognized
??????????????????????????????????????????????????????????????????????????????
Solution 1: Python might not be in PATH
  • Reinstall Python
  • Check "Add Python to PATH" during installation

Solution 2: Try using python3 instead
  python3 app.py

Solution 3: Use full path to Python
  "C:\Program Files\Python39\python.exe" app.py


ERROR: ModuleNotFoundError: No module named 'flask'
??????????????????????????????????????????????????????????????????????????????
Solution: Install requirements
  pip install -r requirements.txt


ERROR: Address already in use (port 5000)
??????????????????????????????????????????????????????????????????????????????
Solution 1: Stop other Flask instances
  Press: Ctrl + C in all Command Prompt windows

Solution 2: Kill the process on port 5000
  netstat -ano | findstr :5000
  taskkill /PID <PID> /F

Solution 3: Use a different port
  Change in app.py:
  app.run(debug=True, port=5001)


ERROR: Camera not found
??????????????????????????????????????????????????????????????????????????????
Solution:
  • Check camera is connected
  • Close other camera apps
  • Restart the application


ERROR: Permission denied
??????????????????????????????????????????????????????????????????????????????
Solution:
  • Run Command Prompt as Administrator
  • Right-click cmd.exe ? "Run as Administrator"

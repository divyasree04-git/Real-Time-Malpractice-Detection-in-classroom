                      ?? TROUBLESHOOTING
?????????????????????????????????????????????????????????????????????????????

Q: Python not found?
A: Install Python 3.7+ from https://www.python.org/
   Check "Add Python to PATH" during installation

Q: Webcam not working?
A: Ģ Check if camera is connected
   Ģ Close other apps using camera
   Ģ Grant camera permissions if prompted

Q: Port 5000 in use?
A: Edit app.py (last line)
   Change: app.run(debug=True, port=5001)

Q: Installation hangs?
A: Press Ctrl+C, then:
   pip install --upgrade pip
   Try again

Q: Database issues?
A: Delete users.db file
   Application auto-recreates it
